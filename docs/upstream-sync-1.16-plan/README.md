# TauriTavern 上游 1.16.0 同步计划（先规划，后实施）

## 1. 目标

在不引入冗余实现的前提下，将 `TauriTavern` 从上游 `SillyTavern 1.15.x` 基线同步到 `1.16.0`，保持：

- 前端文件系统与上游一致（便于用户迁移）
- Tauri 注入层可维护、低耦合
- Rust 后端命令与前端路由契约稳定

本轮仅输出计划，不改业务代码。
